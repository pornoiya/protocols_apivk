Задача 8. VK API.
Приложение, которое выдает список друзей указанного пользователя.
создатель @pornoiya

ОПИСАНИЕ:
    Чтобы пользоваться апи вк, нужно зарегестрировать приложение.
    Если у вас его нет, следуйте инструкции: https://vk.com/dev/first_guide.
    После успешной регистрации вы будете иметь токен и id вашего приложения.
        -aid -- id вашего приложения
        -uid -- id пользователя, чей список друзей нужно показать
        -tkn -- ваш токен
    Отмечу, что, если в порцессе регистрации вы не указывали scope=offline, то
    токен будет генерироваться для одного IP адреса. Т.е., если у вас динамический
    IP, то воспользоваться токеном вы сможете только в течение текущей сессии, а
    при смене IP нужно будет поменять токен по той же инструкции.

    Если вы ввели несуществуещего пользователя, будет выброшен KeyError, если вы
    хотите узнать список друзей пользователя, чей профиль приватный, будет выброшен
    KeyError.

ЗАПУСК:
    >> user.py -uid "pornoiya" -aid "app_id" -tkn "your_secret_token"
    OR
    >> user.py -uid "id455701647" -aid "app_id" -tkn "your_secret_token"
    OR
    >> user.py -uid "455701647" -aid "app_id" -tkn "your_secret_token"
